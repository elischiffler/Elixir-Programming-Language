# Command to set up elixir file

This is because Elixir uses something called "mix" to manage projects

``` shell
mix new szmx_interpreter
```

Our logic lives in szmx_interpreter/lib/szmx_interpreter.ex


# Running/Testing

``` shell
cd szmx_interpreter
mix test
```
